# SwarmAlpha 项目摘要

SwarmAlpha 是一个**可嵌入的多智能体认知治理运行时**——不替代现有框架，而是作为插件层嵌入 AutoGen、CrewAI、LangGraph 等系统，提供实时群体决策的观测、检测与干预能力。

**定位**：Agent 治理分为两层。安全层（Microsoft Agent Governance Toolkit、ACS 标准）防止 agent 做坏事——越权调用工具、超预算、泄露数据。认知层（SwarmAlpha）防止 agent 想错事——回声室、权威偏差、极化、过早共识。两层互补，完整的 agent 治理栈需要两者。上海交大 Li et al. (2026) 独立证实了 MAS 回声室放大偏差且标准检测方法漏检 [arXiv:2604.08963](https://arxiv.org/abs/2604.08963)。SwarmAlpha 通过 `StateInferenceBridge` 兼容 ACS 标准中间件 hook。

**应用场景**：为 LLM 多 agent 协作决策提供实时过程治理——在共识形成过程中检测四类认知偏差并施加干预。

---

## 核心定位：三层独立价值

治理引擎的价值不绑定于"提升最终决策正确率"这一单一指标。三层价值各自独立成立：

| 层次 | 功能 | 独立价值 |
|------|------|----------|
| **过程监控** | 实时检测 4 类群体认知失败（回声室、权威偏差、极化、过早共识） | 即使不干预，"知道 agent 团队正在形成回声室"本身就是价值——企业部署 AI agent 团队需要可观测性 |
| **决策审计** | 完整决策链追溯：谁影响了谁、信念何时转变、治理何时介入 | 事后追责和合规——不改变结果，但改变责任归属能力 |
| **自适应治理** | 检测到偏差后触发目标干预（多样性注入、强制反思、权重调整、继续讨论） | 在真正互依的任务上加速收敛——但有明确边界条件 |

## 架构原则

LLM 仅做感知（自然语言→信念提取）。所有治理逻辑——共识计算、偏差检测、信念动力学——使用纯数学。结果是**快速、廉价、可解释**——嵌入即用，近乎零额外 LLM 调用（仅当 agent 未输出 `[GOV]` 标签时回退到 LLM 推断）。

核心引擎经框架无关性自验证：4 种检测器、4 种干预策略、自适应阈值全部零依赖 DiscussionEngine。通过 `StateInferenceBridge` 的 prompt 注入机制，任一框架 2-4 小时可出原型。

**社会热力学 F 分解驱动的干预优先级**：当多个检测器同时触发时（Crisis 实验中 91.7% 的场景），按自由能 F = (1-R) + T·H 的分解项对干预排序。回测证伪了原假设 `force_reflection↔结构性无序`（p=0.041：极化时 Δτ=−0.033 有害，热性主导时 Δτ=+0.115 有益），已修正为 `force_reflection↔thermal·(1-structural)`（降噪干预，极化时降权）；`reduce_weight↔热性无序` 方向支持。此前干预按硬编码顺序执行，现在按系统"物理状态"排序。

## 因果效应估计

超越相关性分析（t 检验、置换检验），SwarmAlpha 引入**因果效应估计模块**——回答"干预导致最终 τ 变化了多少，相比无干预的反事实？"

方法：最近邻轨迹匹配（k=5）+ 逆距离加权反事实估计 + 10000 次置换检验 + 10000 次 Bootstrap CI。对每个处理实验，从 None 基线池中按第 1 轮轨迹距离选 5 个最近邻 donor，反事实 τ = 匹配 donor 最终 τ 的加权平均。

在 165 次实验数据上的关键发现（注意：数据早于 2026-07-12 治理环路修复，state-modification 类干预效应可能被低估）：

- **M&A 5 轮**：因果效应 +0.135，95% CI [+0.07, +0.20] 不含 0，d=0.96（大效应），p=0.067——所有分析中最接近显著的结果
- **Invest 3 轮**：效应 +0.193，CI 下界刚好 >0，d=0.69（中等）——与边界条件假设一致
- **Invest 5 轮**：效应 −0.111，方向为负——与"多轮治理有害"发现一致

## 实验发现：治理有效

### 2026-07-14 验证实验（Crisis 任务，72 次，扩样确认）

**实验设计改进**：
- 新任务：危机响应优先级排序（5 方案 × 5 维度），消除信息泄露
- 修复 D1-D4 治理环路断裂（信念注入、顺序发言、显式引用、prompt 注入）
- 修复 API 失败导致的假收敛（重试 + 跳过而非返回默认值）
- 3 轮讨论，给治理留出干预空间

**结果**（n=24/cell，全部 rounds=3）：

| 模式 | τ 均值±σ | Cohen's d vs none | p 值 | 功效 |
|------|---------|-------------------|------|------|
| none (基线) | 0.408 ± 0.182 | — | — | — |
| full (治理) | 0.617 ± 0.263 | **0.92** | **0.005** | 88% ✅ |
| shuffle (洗牌) | 0.717 ± 0.243 | **1.44** | <0.001 | 100% |

**关键发现**：
1. **治理统计确认有效**：full vs none 大正效应（d=0.92, p=0.005, 功效 88%），τ 提升 51%
2. **洗牌仍最强**：d=1.44，打破专业知识垄断是最有效干预。shuffle 代表信息交换的理论天花板——所有 agent 获得全部专业知识时的决策质量上限，不是 agent 协作的天花板
3. **基线合理**：τ=0.408 远低于天花板，治理有充足改善空间
4. **干预激活**：89 次干预触发，47 次有效（52.8%）

### 干预有效性与成本分析

对 89 次干预逐条追溯目标 agent 信念变化与决策质量影响：

**有效率与决策质量**：
- 有效干预（47 次）：τ 平均变化 +0.265，确切提升决策质量
- 无效干预（42 次）：τ 平均变化 +0.060，不降低决策质量

**四类干预性价比**：

| 干预类型 | 有效率 | 平均τ变化 | 平均token成本 | 结论 |
|----------|--------|-----------|--------------|------|
| force_reflection | 79.4% | +0.222 | 1,295 | 最可靠 |
| reduce_weight | 61.3% | +0.389 | 1,100 | 性价比最高 |
| introduce_diversity | 9.1% | +0.000 | 880 | 几乎无效 |
| continue_discussion | 0.0% | -0.400 | 1,025 | 有害 |

**机制消融统计检验**（vs 无干预基线 Δτ）：

| 干预类型 | Crisis d | Crisis p | Supplier d | 跨任务一致性 |
|----------|----------|----------|------------|-------------|
| reduce_weight | **+1.51** | **0.0001** | +0.25 | ✅ 方向一致 |
| force_reflection | **+0.73** | **0.001** | +0.04 | ✅ 方向一致 |

**干预时机**：第1轮 82.5% 有效，第2轮 62.2%，第3轮 0.0%（最后一轮无法改变结果）。

**Token 成本**：总 75,680 tokens（约 5,045/实验），其中 50.9% 浪费在无效干预上。停用有害/无效干预 + 第3轮停止干预，可节省 66.0% 成本。

> **局限性**：token 成本为估算值；第3轮 0% 有效率部分受测量限制；仅单一任务类型验证机制消融显著性。

### 2026-07-13 历史实验（Invest/M&A 任务，165 次）

**注意**：数据早于治理环路修复，state-modification 类干预效应被低估。

**关键发现**：
- **M&A 5 轮**：因果效应 +0.135，95% CI [+0.07, +0.20]，d=0.96，p=0.067
- **Invest 3 轮**：效应 +0.193，CI 下界刚好 >0，d=0.69
- **Invest 5 轮**：效应 −0.111，方向为负——与"多轮治理有害"发现一致
- **Shuffle 对照**：M&A 洗牌显著（p=0.0009，d=+1.80）

**自纠错**：初始实验受 system prompt 答案泄露影响，导致基线 τ=1.000。修复后重新设计 Crisis 任务验证。

## 长期方向：智能体社会的基础设施

项目代码已实现 Flat/Grouped/Committee 三层拓扑，引擎 agent 数量无关。多智能体社会是 AI 的下一个前沿，而所有人都在做"如何模拟"，没人在做"如何治理"。SwarmAlpha 的观测→建模→检测→干预→评估循环是未来 AI 社会治理的极简内核。
