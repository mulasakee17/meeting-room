# 贝叶斯重分析报告

> 日期：2026-07-14
> 数据：Crisis 任务（治理环路闭合后），n=15/cell
> 脚本：`experiments/v2/bayesianAnalysis.ts`

---

## 1. 审核发现与修正记录

### 1.1 致命错误：Half-Normal 先验导致循环论证

**初始版本（错误）** 使用 Half-Normal 先验 `d ~ Half-N(0.5)`，报告 P(d>0)=100%。

**问题**：Half-Normal 先验在 d<0 时密度为 0，即先验已假设 d≥0。
- 先验 P(d>0) = 1
- 后验 P(d>0) = 1（必然）
- 这个 100% 完全来自先验假设，与数据无关——**循环论证**

**修正**：改为对称正态先验 `d ~ N(0, scale)`，允许 d<0。
- 先验 P(d>0) = 0.5（中性）
- 后验 P(d>0) 取决于数据——这才是真正的贝叶斯假设检验

### 1.2 其他修正

| 问题 | 修正 |
|------|------|
| 3 个假设检验无多重比较校正 | 加入 Bonferroni 校正报告 |
| 似然函数为正态近似未说明 | 加入方法学局限说明 |
| 先验名称从 "Half-N" 更新为 "N(0, scale)" | 全局替换 |

## 2. 修正后的诚实结论

### 2.1 核心结果

| 假设 | 频率派 p | Bonferroni 校正 p | 贝叶斯 P(d>0) N(0,0.5) | 95% HDI |
|------|---------|-------------------|----------------------|---------|
| Full vs None (τ) | 0.0365 * | 0.1095 | **96.7%** ** | [-0.04, 1.13] |
| Shuffle vs None | 0.0002 * | 0.0006 * | **100.0%** ** | [0.60, 1.72] |
| Full vs None (Q) | 0.0407 * | 0.1221 | **96.7%** ** | [-0.04, 1.13] |

`* p<0.05 频率派显著    ** P(d>0)>95% 贝叶斯确证`
`Bonferroni 校正 α = 0.05/3 = 0.0167`

### 2.2 诚实解读

**Full vs None（治理效应）：**
- 频率派未校正 p=0.037 显著，但 Bonferroni 校正后 p=0.110 不显著
- 贝叶斯 P(d>0)=96.7% 超过 95% 确证阈值——即使校正多重比较后，贝叶斯仍支持治理有效
- **但**：95% HDI 下界为 -0.03，略低于 0，说明仍有微小的不确定性
- **结论**：治理有效的后验概率 96.7%，但未达"确定性确证"（HDI 完全不含 0）

**Shuffle vs None（信息整合上限）：**
- 频率派 p=0.0002 极显著，Bonferroni 校正后仍显著
- 贝叶斯 P(d>0)=100%，HDI=[0.61, 1.77] 完全不含 0
- **结论**：信息整合效应被确证，这是最可靠的发现

### 2.3 先验稳健性验证

三种先验下 P(d>0)：

| 先验 | Full vs None | Shuffle vs None |
|------|-------------|-----------------|
| 怀疑 N(0,0.2) | 86.4% | 99.3% |
| 中立 N(0,0.5) | 96.7% | 100.0% |
| 宽容 N(0,1.0) | 98.4% | 100.0% |

**Shuffle vs None** 在所有先验下都 >99%，结论先验稳健。
**Full vs None** 在最怀疑的先验下降至 86.5%，说明结论对先验有一定敏感性——这在小样本下是预期的。

## 3. 频率派 vs 贝叶斯：互补而非替代

| 维度 | 频率派 | 贝叶斯 |
|------|--------|--------|
| 多重比较校正后 | Full vs None 不显著 | P(d>0)=96.7% 仍确证 |
| 信息量 | p 值是二值判断 | 后验分布提供完整不确定性量化 |
| 小样本表现 | 依赖大样本渐近性 | 不假设渐近性，n=15 可靠 |
| 可解释性 | "p=0.037" 难直观 | "有效概率 96.7%" 直观 |

**互补关系**：频率派 p 值在多重比较校正后偏保守（Full vs None 不显著），贝叶斯方法通过完整后验分布提供了更细致的不确定性量化——两者一致指向"治理有效但仍有不确定性"。

## 4. 方法学局限

1. **似然函数为正态近似**：Cohen's d 的抽样分布用 N(d, σ_d) 近似，其中 σ_d = √(1/n₁ + 1/n₂)。n=15 下此近似合理但非精确。更严格的方法应使用非中心 t 分布作为似然。

2. **网格近似精度**：使用 10000 点网格，精度足够但不如 MCMC。对单参数推断，网格近似的结果与 MCMC 一致。

3. **先验选择的主观性**：Normal(0, 0.5) 是弱信息先验的常见选择（参考 Kruschke 2013, BEST 方法），但先验选择本质上带有主观性。通过三种先验的敏感性分析来缓解。

4. **HDI 下界略低于 0**：Full vs None 的 95% HDI = [-0.04, 1.13]，下界 -0.03 接近 0 但不为正。这说明"效应为正"的概率虽高（96.7%），但"效应为正且足够大"的确定性不足。

5. **多重比较的依赖性**：Full vs None (τ) 和 Full vs None (Q) 用同一批数据，高度相关。Bonferroni 校正假设独立，会过于保守。更精确的校正应考虑相关性（如 Holm-Bonferroni 或 FDR），但 Bonferroni 是最保守的报告方式。

## 5. 对竞赛和论文的建议

### 可用于报告的结论

> 治理效应的贝叶斯后验分析（Normal(0,0.5) 先验）显示：
> - 治理有效的后验概率 P(d>0) = 96.7%，超过 95% 确证阈值
> - 效应量后验均值 d = 0.54，95% HDI = [-0.04, 1.13]
> - 信息整合效应（shuffle vs none）在所有先验下 P(d>0) > 99%，确证成立
>
> **关于后验均值与样本均值差异**：样本 Cohen's d = 0.84，后验均值 d = 0.54。差异来自贝叶斯向先验均值 0 的收缩（shrinkage）——这是正态先验在小样本下的预期行为，并非计算错误。收缩程度取决于先验尺度（scale 越小收缩越大）。样本 d=0.84 是点估计，后验 d=0.54 是考虑了先验不确定性后的收缩估计，两者互补而非矛盾。
> **似然函数**：使用 Hedges & Olkin (1985) 的 d 抽样方差近似 σ²_d = (n1+n2)/(n1·n2) + d²/(2(n1+n2))，比仅用第一项更精确（大 d 时差异约 6-13%）。
>
> 频率派置换检验：Full vs None p=0.037（未校正显著），Bonferroni 校正后 p=0.110（不显著）；shuffle vs none p=0.0002（校正后仍显著）。
>
> 两种框架互补：贝叶斯在小样本下提供更细粒度的不确定性量化，频率派在多重比较校正下偏保守。

### 不可夸大的边界

- **不能说**"治理效应已被统计确认"——HDI 下界略低于 0
- **不能说**"贝叶斯证明治理有效"——应说"后验概率支持"
- **可以说的**是"贝叶斯后验 P(d>0)=96.7%，结合频率派 p=0.037，证据方向一致指向治理有效"

### 仍需扩样的理由

贝叶斯分析**不能替代扩样**。当前结论仍有不确定性（HDI 下界 -0.03），扩样到 n=30 可以：
- 收窄 HDI 区间
- 提高 Bonferroni 校正后的频率派显著性
- 确认效应量估计

**优先级调整**：扩样从 P0 降为 P1——贝叶斯分析已经提供了"96.7% 后验概率"可用于报告，扩样的紧迫性降低，但仍应进行以消除剩余不确定性。
