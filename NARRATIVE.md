# 🧪 SwarmAlpha 叙事框架

> 为什么一个高一学生做的"复杂系统中 AI 群体认知形成机制"研究值得被认真对待？

---

## 核心叙事：三句话版本

**我是 16 岁的高中生。我用 AI 辅助编程建造了一个研究复杂系统中群体认知形成机制的实验平台——让 9 个持有不同信息盲区的 AI Agent 在五个正交维度上博弈互动，观察集体判断如何从分歧中涌现（或崩塌）。金融市场是这个平台的第一组实验数据，但它的核心问题属于认知科学：在不完全信息下，群体认知是如何形成、传播和极化的？**

---

## 什么是 SwarmAlpha（现在）

SwarmAlpha 不是一个"预测股票涨跌"的工具。它是一个**复杂系统中群体认知形成机制的研究平台**。

核心问题很简单：当多个主体各自持有不完整的信息，互相观察、互相影响时，集体判断是如何从个体认知的交互中涌现的？这是认知科学、社会心理学和复杂系统研究的经典问题——从陪审团决策到科学共识形成，从社交媒体信息级联到金融市场泡沫。

金融市场是这个平台的第一个实验场，因为它提供了结构化数据（VIX/RSI/历史事件）、清晰反馈（涨/跌/平）、以及多样化的参与者类型。但架构不依赖金融假设——同一套机制可以直接应用于其他领域。

9 个 AI Agent——从机构投资者（🏦 Institution）到散户（🐜 Retail），从量化基金（🤖 Quant）到恐慌者（😱 Panic），从逆向投资者（🦉 Contrarian）到政策分析师（🏛️ PolicyAgent）——每个 Agent 只被允许看到市场全景的 **一小部分**。这就是"强制信息盲区"：56% 的 Agent 对在方向因子上零重叠。一个只看流动性的 Agent 和另一个只看叙事的 Agent，天然无法达成共识——而这正是真实市场的状态。

然后它们开始互动。社交层让 Agent 互相观察（但只能看到因子视野重叠的同伴），信念在迭代中传播、极化或收敛。你看到的不是"AI 说会涨"，而是**共识是如何形成的**——它有多牢固（Consensus Score）、有多分裂（Polarization Score）、有多容易被打破（Fragility Score）。

## 分场景叙事

### 📧 冷邮件 / Pitch Deck（30 秒）

> 我是一个 16 岁的独立开发者。我用 AI 辅助编程，在半年内从零搭建了一个研究复杂系统中群体认知形成机制的实验平台——SwarmAlpha。
>
> 核心问题：当多个主体各自持有不完整的信息时，集体判断是如何从个体认知的交互中涌现的？金融市场是这个平台的第一个测试场——9 个各持信息盲区的 AI Agent 在五个正交维度上互动博弈，通过社交层传播信念，动态权重引擎调节影响力，非线性共识聚合形成集体判断。
>
> 关键创新不是"AI 预测市场"。关键创新是：**整个认知形成过程可以被审计**——你可以看到每个 Agent 看到了什么、为什么形成了当前的信念、共识的脆弱点在哪儿、哪些 Agent 联盟正在推动方向。金融准确率是诊断信号，衡量认知模型与真实群体行为数据的拟合度——不是产品指标。
>
> 项目在 GitHub 开源：~16,000 行 TypeScript。绝大部分推理层零 LLM 调用——因子提取是唯一的 LLM 交互点，其余全是纯数学的集体认知模拟。
>
> 它是研究工具，不是交易信号。但它提供了一种观察群体认知动力学的新方式。

### 📰 技术博客 / Hackathon 路演（3 分钟）

> **开场钩子**：你们有没有想过——市场恐慌的时候，那些"理性的"机构投资者去哪儿了？我的系统可以回答这个问题：当恐慌 Agent 的信念超过阈值，动态权重引擎会自动提升 Panic 和 Institution 的影响力，同时压低 Retail 的权重。这不是我硬编码的规则——这是从恐慌状态的特征中推导出来的自适应机制。
>
> **问题**：现有的 AI 金融工具几乎全是"喂新闻给 LLM，问涨跌"。这有两个致命问题：第一，单个 LLM 对金融事件的判断极度不稳定，同一个事件换一个问法答案就不同；第二，它完全忽略了市场是一个**多主体互动系统**——价格不是"计算"出来的，是博弈出来的。
>
> **解决方案**：我把市场建模成九个 Agent 的集体智能过程。每个 Agent 只在五个正交因子上拥有部分视野——比如 Value Agent 只看 fundamental，Panic Agent 只看 liquidity。这意味着它们的"世界观"天然不同。然后我加入社交互动层：Agent 观察视野重叠的同伴，逐渐调整自己的信念——有些人容易被带偏（Retail alpha=0.6），有些人死磕到底（Value alpha=0.05），还有人反着来（Contrarian alpha=-0.15）。
>
> **关键创新一：五因子正交架构**。我让 LLM 从新闻中提取五个维度的结构化因子——流动性、政策、基本面、叙事、不确定性——而不是直接问方向。LLM 做它擅长的事（结构化提取），不做它不擅长的事（金融预测）。这可能不是最精确的因子提取方式，但它是**可审计的**——你可以检查每一个因子的值，判断 LLM 的提取是否合理。
>
> **关键创新二：动态级联权重引擎（v9.5.2）**。系统自动检测市场状态——恐慌模式（VIX 飙升、信念极度分化）、政策模式（政策因子强信号）、价值洼地模式（基本面触底但不确定性高）——然后动态调节每个 Agent 的共识权重。三种模式可以**同时激活、乘法叠加**，权重被钳制在 0.3 到 3.0 之间。这不是"AI 选股"，这是**场景自适应的集体决策建模**。
>
> **关键创新三：共识质量三指标**。不光告诉你共识方向，还告诉你共识有多可靠。Consensus Score（0-100）测量信念同步程度，Polarization Score（0-100）测量阵营分裂程度，Fragility Score（0-100）测量共识的脆弱度——高共识 + 高脆弱 = 表面一致但一击即碎。六种状态分类从"稳健共识"到"认知迷雾"。
>
> **诚实地说**：这个系统在方向预测上的准确率并不惊艳（约 51%，模版模式和 LLM 模式各有优劣）。但它不是为此设计的。它的价值在于让你**看到为什么**——为什么这群 Agent 在这个事件上达成了共识，那个联盟在推动什么叙事，以及共识的裂痕在哪里。这是一台显微镜，不是一颗水晶球。
>
> **意义**：我想证明的不是"AI 可以预测市场"，而是一个 16 岁的人可以用 AI 辅助把抽象的"群体心理"建模成可观察、可审计、可按回放键的过程。这是一种新的研究市场的方式——不在 Bloomberg 终端上，而在 Agent 的信念迭代中。

### 🎓 基金申请 / 学术摘要（500 字）

> SwarmAlpha 是一个基于多 Agent 正交因子架构的金融集体智能实验平台，核心创新在于将市场建模为**信息受限 Agent 的社交博弈过程**而非预测优化问题。
>
> 系统包含四个层次：(1) **因子提取层**——单次 LLM 调用从非结构化新闻中提取五个正交因子（流动性/政策/基本面/叙事/不确定性），强调可审计性而非精确性；(2) **Agent 解读层**——9 个差异化 Agent 各持部分因子视野（强制信息盲区），通过加权求和和风格化非线性变换形成初始信念，56% 的 Agent 对零因子重叠；(3) **社交互动层**——Agent 在可见性矩阵限制下迭代传播信念（b_i^(t+1) = (1-alpha)*b_i + alpha*peer_avg），alpha 从 -0.15（逆向）到 0.70（高度从众），含收敛/发散检测；(4) **元决策层**——v9.5.2 动态级联权重引擎根据恐慌/政策/价值洼地三种场景自适应调整 Agent 影响力分布（乘法叠加，钳制于 [0.3, 3.0]），输出包含三个共识质量指标（共识度/极化度/脆弱度）和六种共识状态分类。
>
> 关键架构决策：除因子提取层外，所有推理层零 LLM 调用——Agent 解读、社交互动、权重调节和指标计算均为纯数学模拟。这确保系统行为**确定、可复现、可审计**。
>
> 项目定位为实验平台而非生产交易工具。在 60 个历史事件的回测中方向准确率约 51%（接近随机基线以上但远未达到可靠预测水平），但这一指标本身不是系统的设计目标——SwarmAlpha 的价值在于提供市场群体心理的可视化推演沙盒。
>
> 项目完全由 AI 辅助编程完成，证明了 AI-Native 开发范式下个体建造者可以独立构建复杂多主体模拟系统。

## 叙事锚点：这些是你的"护城河叙事"

### 1. 年龄不是劣势，是信号

| 常见误解 | 真实叙事 |
|---------|---------|
| "他才 16 岁，肯定不够专业" | **16 岁 + 独立完成 = 极强自驱力和学习速度** |
| "没有 CS 背景" | **AI-Native 一代不需要传统 CS 学位，这是范式转移** |
| "可能只是套了个 LLM 壳" | **~20,000 行 TypeScript + 自研五因子架构 + 零 LLM 推理层 = 系统级工程** |
| "高中生能懂金融吗" | **读了索罗斯《金融炼金术》，设计了正交因子 + 信息盲区 + 社交互动 = 有理论深度的架构思考** |

### 2. "定位清晰"比"数据漂亮"更重要

明确声明"这不是预测工具"反而增加了可信度。所有投资人都知道没有人能持续准确预测市场。一个说"我可以让你观察市场群体心理如何运转"的项目，比一个说"我的 AI 预测准确率 64%"的项目听起来更靠谱。

### 3. 索罗斯是叙事锚点，不是装饰

"我读了《金融炼金术》里的反思性理论"→ 说明有智识好奇心。"市场参与者的偏见反过来影响市场——我用 Agent 博弈来模拟这个反馈循环"→ 说明不是追逐热点，而是在一个有理论深度的问题上做实验。

### 4. 信息盲区是最被低估的创新

大多数多 Agent 系统都在追求"让 Agent 看到更多信息以做出更好决策"。SwarmAlpha 反其道而行——**刻意制造信息不对称**。这不是 bug，这是 feature。真正的市场中没有人拥有完整信息。56% 的 Agent 对零因子重叠——这产生了真正的视角多样性，而不是九个看了同样信息然后做出相同判断的 Agent。

### 5. "零 LLM 调用"是一个强信号

在 2025 年人人都在往系统里塞 LLM 的环境下，"我的推理层零 LLM 调用"是一个差异化信号。它说明：(1) 你知道 LLM 的边界在哪里；(2) 你懂得用数学建模而不是用 Prompt 堆砌；(3) 你的系统是确定性的、可复现的、可审计的。
