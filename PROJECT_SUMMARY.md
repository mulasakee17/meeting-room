# SwarmAlpha 项目摘要

SwarmAlpha 是一个**可嵌入的多智能体治理运行时**——不替代现有框架，而是作为插件层嵌入 AutoGen、CrewAI、LangGraph 等系统，提供实时群体决策的观测、检测与干预能力。

**应用场景**：为 LLM 多 agent 协作决策提供实时过程治理——在共识形成过程中检测极化、权威偏置、回声室、过早共识四类偏差并施加干预，确保少轮次讨论的决策质量。

---

## 核心定位：三层独立价值

治理引擎的价值不绑定于"提升最终决策正确率"这一单一指标。三层价值各自独立成立：

| 层次 | 功能 | 独立价值 |
|------|------|----------|
| **过程监控** | 实时检测 4 类群体认知失败（回声室、权威偏差、极化、过早共识） | 即使不干预，"知道 agent 团队正在形成回声室"本身就是价值——企业部署 AI agent 团队需要可观测性 |
| **决策审计** | 完整决策链追溯：谁影响了谁、信念何时转变、治理何时介入 | 事后追责和合规——不改变结果，但改变责任归属能力 |
| **自适应治理** | 检测到偏差后触发目标干预（多样性注入、强制反思、权重调整、继续讨论） | 在真正互依的任务上加速收敛——但有明确边界条件 |

## 架构原则

LLM 仅做感知（自然语言→信念提取）。所有治理逻辑——共识计算、偏差检测、信念动力学——使用纯数学。结果是**快速、廉价、可解释**——嵌入即用，近乎零额外 LLM 调用（仅当 agent 未输出 `[GOV]` 标签时回退到 LLM 推断）。

核心引擎经框架无关性自验证：4 种检测器、4 种干预策略、自适应阈值全部零依赖 DiscussionEngine。通过 `StateInferenceBridge` 的 prompt 注入机制，任一框架 2-4 小时可出原型。

## 因果效应估计

超越相关性分析（t 检验、置换检验），SwarmAlpha 引入**因果效应估计模块**——回答"干预导致最终 τ 变化了多少，相比无干预的反事实？"

方法：最近邻轨迹匹配（k=5）+ 逆距离加权反事实估计 + 10000 次置换检验 + 10000 次 Bootstrap CI。对每个处理实验，从 None 基线池中按第 1 轮轨迹距离选 5 个最近邻 donor，反事实 τ = 匹配 donor 最终 τ 的加权平均。

在 165 次实验数据上的关键发现（注意：数据早于 2026-07-12 治理环路修复，state-modification 类干预效应可能被低估）：

- **M&A 5 轮**：因果效应 +0.135，95% CI [+0.07, +0.20] 不含 0，d=0.96（大效应），p=0.067——所有分析中最接近显著的结果
- **Invest 3 轮**：效应 +0.193，CI 下界刚好 >0，d=0.69（中等）——与边界条件假设一致
- **Invest 5 轮**：效应 −0.111，方向为负——与"多轮治理有害"发现一致

## 实验发现：治理有边界

165 次对照实验（2×2 因子设计，n=15/cell）：

- **弱互依任务**（M&A）：全治理 vs 无治理差异不显著（p=0.36）。仅 Shuffle 对照显著（p=0.0009）——打破专业过度自信比精准干预更有效。
- **强互依任务**（Invest）：3 轮有方向性改善（d=+0.65，p=0.152，不显著）；5 轮完全归零（d=+0.00）。强制反思显著有害（p=0.048）。
- **自纠错**：初始实验结果受 system prompt 答案泄漏影响。独立发现并修复全部 6 个硬伤后，叙事从"治理有效"更新为"治理有边界"——这一过程本身就是学术诚信的证明。

## 长期方向：智能体社会的基础设施

项目代码已实现 Flat/Grouped/Committee 三层拓扑，引擎 agent 数量无关。多智能体社会是 AI 的下一个前沿，而所有人都在做"如何模拟"，没人在做"如何治理"。SwarmAlpha 的观测→建模→检测→干预→评估循环是未来 AI 社会治理的极简内核。
